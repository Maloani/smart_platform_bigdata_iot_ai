# Smart Platform Big Data IoT AI

## Description
Projet complet intégrant :
- Kafka (ingestion)
- Spark Streaming (traitement)
- LSTM (IA)
- TensorFlow Lite (Edge AI)

## Lancer le projet

### 1. Kafka
python kafka_pipeline/producer.py
python kafka_pipeline/consumer.py

### 2. Spark
python spark_streaming/streaming.py

### 3. IA
python ml_models/train_model.py

### 4. Edge AI
python edge_ai/convert_tflite.py
python edge_ai/inference.py
